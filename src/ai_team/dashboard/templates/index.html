Help & FAQ

Install the project, then launch the dashboard from your project directory:

pip install -e /path/to/ai-team cd /your/project && ai-team serve

The dashboard starts at http://127.0.0.1:8420. From here you can monitor agents, view the task board, run pipelines, and chat with agents.

  • feature_dev — Full 6-step lifecycle: PM → Researcher → Architect → Coder → Tester → Reviewer
  • bugfix — 4-step flow: Researcher → Coder → Tester → Reviewer
  • code_review — 3-step review: Reviewer → Coder → Reviewer

Run a pipeline from the CLI:

ai-team run --pipeline feature_dev --goal "Add dark mode"

Or use the Submit Goal bar in the dashboard.

  • PM — Decomposes goals into tasks, prioritizes, and tracks progress
  • Researcher — Gathers context, reads the codebase, searches docs
  • Architect — Designs technical solutions and writes architecture docs
  • Coder — Implements code on feature branches and commits changes
  • Tester — Writes and runs tests, measures coverage
  • Reviewer — Reviews code quality, security, and suggests improvements
  • Stats Bar — Shows agents online, active tasks, blocked tasks, active groups, and events
  • Goal Bar — Submit a new goal to create a task group
  • Filter Bar — Filter tasks by group, role, status, and priority
  • Board View — Kanban view: Blocked, Pending, In Progress, Completed, Rejected
  • List View — Sortable table of all tasks
  • Graph View — DAG visualization of task dependencies within a group
  • Agent Sidebar — View agent statuses, chat with agents
  • Event Log — Real-time stream of all system events
ai-team serve

Start the dashboard server (default: port 8420)

ai-team run --pipeline <name> --goal "description"

Execute a pipeline with a goal. Pipelines: feature_dev, bugfix, code_review

ai-team status

Check the current status of agents and tasks
