Help & FAQ

Install the project, then launch the dashboard and agents:

pip install -e /path/to/ai-team cd /your/project && ai-team serve

The dashboard starts at http://127.0.0.1:8420. Agents launch automatically based on your config/roles/ YAML files. Submit a goal via the Goal Bar or CLI to kick off work.

There are no fixed pipelines. Agents create tasks for each other based on configurable routing rules:

  • Groups — Every initiative gets a group ID (e.g. FEAT-001). All downstream tasks share this ID for audit.
  • Task Flow — PM creates PRD → tasks for Architect → Architect creates tasks for Coder → Coder creates Tester + Reviewer tasks (chained)
  • Dependencies — Tasks can be blocked by other tasks. When a dependency completes, blocked tasks auto-unblock.
  • Rejections — Reviewers can reject work, creating a new linked revision task back to the original author.

Submit a goal from the CLI:

ai-team goal "Add dark mode to the application"

Roles are defined in config/roles/*.yaml. Default roles:

  • PM — Receives goals, creates PRDs, routes to Architect. Creates FEAT groups.
  • Architect — Designs technical solutions, creates implementation tasks for Coder. Can create DEBT groups independently.
  • Coder — Implements code, creates chained Tester + Reviewer tasks. Scales up to 3 instances.
  • Tester — Runs tests and verifies implementations. Completing a test auto-unblocks the Reviewer.
  • Reviewer — Reviews code quality. Can approve (complete) or reject (creates revision task).

Add new roles by creating a YAML file in config/roles/ and updating routing. No code changes needed.

  • Stats Bar — Shows agents online, active tasks, blocked tasks, active groups, and events
  • Goal Bar — Submit a new goal to create a task group assigned to PM
  • Filter Bar — Filter tasks by group, role, status, and priority
  • Board View — Kanban columns: Blocked, Pending, In Progress, Completed, Rejected
  • List View — Sortable table of all tasks with inline details
  • Graph View — DAG visualization of task dependencies within a group for end-to-end audit
  • Agent Sidebar — View agent instance statuses, chat with agents
  • Event Log — Real-time stream of all system events
ai-team serve

Start the dashboard and all agent instances

ai-team goal "Add dark mode"

Submit a new goal (creates a FEAT group and PM task)

ai-team status

Check current status of agents, tasks, and groups
